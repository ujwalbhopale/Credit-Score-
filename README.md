This project focuses on analyzing customer, loan, and credit behavior data to identify factors that contribute to loan default risk.
The goal is to help financial institutions make data-driven, transparent, and explainable credit decisions.

The project involves:
Exploratory Data Analysis (EDA)
Feature engineering
Categorical encoding
Correlation analysis
Business-oriented insights aligned with problem expectations

Objectives
Identify risky loan applicants early
Understand how customer demographics, credit behavior, and loan characteristics impact defaults
Provide explainable insights for credit approval decisions
Prepare a machine-learning-ready dataset
